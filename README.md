## Run Locally:

1. Clone the project

```bash
  git clone https://github.com/
```

2. Go to the project directory

```bash
  cd Personality-Predictor
```

3. Install dependencies

```bash
  pip install -r requirements.txt
```

4. Start the server

```bash
  streamlit run app.py
```
