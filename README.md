## Run Locally:

1. Clone the project

```bash
  git clone https://github.com/ohthatspaul/mbti_predictor
```

2. Go to the project directory

```bash
  cd mbti_predictor
```

3. Install dependencies

```bash
  pip install -r requirements.txt
```

4. Start the server

```bash
  streamlit run app.py
```
